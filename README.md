# AdobeAIR-DiscordRPC

Fork of `AdobeAIR-DiscordRPC`.

---

## Fork Changes

- Adds Windows 64-bit ANE support: `Windows-x86-64`.
- Adds macOS ANE support: `MacOS-x86-64`.
- Keeps original Windows 32-bit support: `Windows-x86`.
- Builds Discord RPC from the official Discord RPC `v3.4.0` release libraries.
- Builds and packages `DiscordRPC.ane` with GitHub Actions.
---

## Release

Run the `Build Extension` workflow manually. It builds `DiscordRPC.ane` and publishes it to the `latest` GitHub Release.
